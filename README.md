Deskripsi singkat : 
Proyek ini adalah Dashboard Administrasi Produk Sederhana yang dibangun menggunakan HTML, CSS, dan Vanilla JavaScript.
Tujuan utamanya adalah menyediakan antarmuka frontend yang modern dan responsif untuk mengelola data produk.
Fitur Utama:
Akses: Dilengkapi halaman Login dengan validasi dasar.
Ringkasan Data: Dashboard menampilkan Kartu Ringkasan (KPI) yang diisi secara dinamis (Total Products, Total Sales, Total Revenue dalam format Rupiah).
Pengelolaan: Halaman terpisah berisi Tabel Data Produk yang diisi secara dinamis, lengkap dengan simulasi tombol aksi Edit dan Hapus.
Desain: Menggunakan tata letak sidebar ikon yang ringkas dan styling yang bersih (menggunakan Poppins dan Flexbox).
